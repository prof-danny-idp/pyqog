Visao Geral dos Datasets

O Quality of Government Institute disponibiliza cinco datasets principais, cada um disponivel nos formatos time-series (ts) e cross-sectional (cs).

Dataset Prefixo Parametro pyqog Descricao
Basic bas "basic" Subconjunto selecionado de variaveis do QoG Standard. Ideal para ensino e analises rapidas.
Standard std "standard" Dataset completo com centenas de variaveis de diversas fontes internacionais. O mais abrangente.
OECD oecd "oecd" Dados especificos para paises membros da OCDE, com variaveis adicionais pertinentes.
Environmental Indicators ei "environmental" Indicadores ambientais e de sustentabilidade de governanca.
Social Policy soc "social_policy" Dados sobre politicas sociais, bem-estar e sistemas de protecao social.

QoG Basic Dataset

Descricao

O QoG Basic Dataset e um subconjunto do QoG Standard, contendo uma selecao das variaveis mais utilizadas em pesquisa comparativa. E ideal para estudantes e para analises exploratorias que nao necessitam do conjunto completo de variaveis.

Prefixo: bas | Formatos: time-series (ts), cross-sectional (cs)

Codebook atual: codebook_bas_jan26.pdf

# Baixar QoG Basic
df = pyqog.read_qog(which_data="basic")                         # time-series
df = pyqog.read_qog(which_data="basic", data_type="cross-sectional")  # cross-sectional

QoG Standard Dataset

Descricao

O QoG Standard Dataset e o dataset mais completo, contendo centenas de variaveis de diversas fontes internacionais como Banco Mundial, ONU, Transparency International, Freedom House, V-Dem, entre muitas outras. Cobre temas como democracia, corrupcao, economia, saude, educacao e meio ambiente.

Prefixo: std | Formatos: time-series (ts), cross-sectional (cs)

Codebook atual: codebook_std_jan26.pdf

# Baixar QoG Standard
df = pyqog.read_qog(which_data="standard")                         # time-series
df = pyqog.read_qog(which_data="standard", data_type="cross-sectional")  # cross-sectional

QoG OECD Dataset

Descricao

O QoG OECD Dataset contem dados especificos para os paises membros da Organizacao para a Cooperacao e Desenvolvimento Economico (OCDE). Inclui variaveis adicionais que so estao disponiveis para esse grupo de paises.

Prefixo: oecd | Formatos: time-series (ts), cross-sectional (cs)

Codebook atual: codebook_oecd_jan26.pdf

# Baixar QoG OECD
df = pyqog.read_qog(which_data="oecd")                         # time-series
df = pyqog.read_qog(which_data="oecd", data_type="cross-sectional")  # cross-sectional

QoG Environmental Indicators

Descricao

O QoG Environmental Indicators Dataset foca em indicadores ambientais e de sustentabilidade, incluindo variaveis sobre emissoes, energia, recursos naturais e politicas ambientais.

Prefixo: ei | Formatos: time-series (ts), cross-sectional (cs)

Codebook atual: codebook_ei_jan26.pdf

# Baixar QoG Environmental
df = pyqog.read_qog(which_data="environmental")                         # time-series
df = pyqog.read_qog(which_data="environmental", data_type="cross-sectional")  # cross-sectional

QoG Social Policy Dataset

Descricao

O QoG Social Policy Dataset contem dados sobre politicas sociais, sistemas de bem-estar, protecao social, gastos sociais e indicadores relacionados.

Prefixo: soc | Formatos: time-series (ts), cross-sectional (cs)

Codebook atual: codebook_soc_jan26.pdf

# Baixar QoG Social Policy
df = pyqog.read_qog(which_data="social_policy")                         # time-series
df = pyqog.read_qog(which_data="social_policy", data_type="cross-sectional")  # cross-sectional

Acessar codebooks via pyqog

import pyqog

# Obter URL do codebook para qualquer dataset e versao
url = pyqog.get_codebook_url("standard", 2026)
print(url)  # https://www.qogdata.pol.gu.se/data/codebook_std_jan26.pdf

url = pyqog.get_codebook_url("basic", 2020)
print(url)  # https://www.qogdata.pol.gu.se/dataarchive/codebook_bas_jan20.pdf
